라벨 템플릿 필드 가이드

Brother P-touch Editor에서 템플릿 생성 시 아래 필드명을 사용하세요.

기본 필드

필드명 설명 예시
patientName 환자명 홍길동
medicineName 약품명 시크렌캡슐
medicineType 약품 종류 먹는약, 바르는약
dose 복용법 2알씩 하루 3번 복용
prescriptionDays 처방일수 7일분
madeDate 조제일 조제일 2025.08.11
pharmacy 약국명 (설정에서 지정) 행복약국

약품 상세 필드

필드명 설명 예시
dailyDose 일일 투여 횟수 3
singleDose 1회 투여량 2
dailyDoseText 일일 투여 텍스트 하루 3회
singleDoseText 1회 투여 텍스트 1회 2정
prescriptionDaysText 처방일수 텍스트 7일분

처방전 필드

필드명 설명 예시
hospitalName 병원명 서울대학교병원
receiptDate 접수일자 2025년08월11일
receiptNum 접수번호 12345

참고: 템플릿 필드명은 대소문자를 정확히 구분해야 합니다. Brother P-touch Editor에서 텍스트 객체의 이름을 위 필드명과 동일하게 설정하세요.
