PM프로그램에서 ATC사용-B타입으로 설정된 경로의 폴더를 선택해주세요
⚠️ 중요:
• ATC를 사용하시는 경우 OCS파일은 자동으로 삭제됩니다. 따라서 "삭제안함"(체크 해제)으로 설정하세요.
• ATC를 사용하지 않거나, JVM의 OCS정보 복제 기능을 사용하는 경우 "자동삭제"(체크)로 설정하면 처방전 리딩 후 OCS파일을 자동으로 삭제합니다.
• 자동으로 삭제하지 않을 경우 파일이 쌓여서 공간을 차지하고, 해당 파일이 개인정보를 담고 있으니 가능하면 자동삭제를 선택하세요.

라벨 템플릿 필드 가이드

Brother P-touch Editor에서 템플릿 생성 시 아래 필드명을 사용하세요.

기본 필드

필드명 설명 예시
patientName 환자명 홍길동
medicineName 약품명 타이레놀정 500mg
medicineType 약품 종류 (제형) 먹는약, 바르는약, 안약
dosageText 복용법 전체 텍스트 하루 3번 아침,점심,저녁 식후 2정씩
receiptDate 처방일 (조제일) 2025년11월05일
unit 약품 단위 정, 캡슐, 포, mL

복용 정보 필드

필드명 설명 예시
prescriptionDays 처방일수 (숫자) 7
singleDose 1회 복용량 (숫자) 2
dailyDose 하루 복용횟수 (숫자) 3
totalAmount 총량 (숫자) 42

처방전 정보 필드

필드명 설명 예시
hospitalName 병원명 서울대학교병원
receiptNum 접수번호 12345
patientId 환자번호 12345678901

참고: 템플릿 필드명은 대소문자를 정확히 구분해야 합니다. Brother P-touch Editor에서 텍스트 객체의 이름을 위 필드명과 동일하게 설정하세요.

⚠️ Brother b-PAC가 설치되지 않았습니다

라벨 출력을 사용하려면 b-PAC Client Component를 설치해주세요.
